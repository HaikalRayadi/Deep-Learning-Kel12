## ⚡ Tentang Proyek

💡 **NeuraShop** merupakan sistem **AI Dynamic Pricing** yang dirancang untuk memprediksi harga optimal secara real-time menggunakan teknologi **Deep Learning**.

Proyek ini dikembangkan sebagai hasil kolaborasi **Kelompok 12** pada mata kuliah **Deep Learning**, dengan tujuan menghadirkan pengalaman penentuan harga yang lebih cerdas, adaptif, dan berbasis data.
